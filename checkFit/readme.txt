Un'app mobile di tracciamento delle attività fitness che consente agli utenti di registrare le loro sessioni di allenamento, monitorare i progressi, impostare obiettivi e visualizzare statistiche dettagliate. L'app include anche funzionalità social per condividere i progressi con amici e partecipare a sfide.

Tecnologie Utilizzate:
    Linguaggio di Programmazione: Dart
    Framework: Flutter
    Backend: Firebase (Firebase Firestore per il database, Firebase Authentication per l'autenticazione, Firebase Storage per la gestione dei file)
    Servizi Esterni: API per il tracciamento GPS
    Design: Material Design
