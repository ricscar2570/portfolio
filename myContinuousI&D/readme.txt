Una pipeline di CI/CD (Continuous Integration/Continuous Deployment) per una applicazione web che include l'integrazione con strumenti di monitoraggio e alerting. Questo progetto dimostra come automatizzare il build, il test, il deploy e il monitoraggio di una applicazione web usando strumenti come Jenkins, Docker, Kubernetes, Prometheus e Grafana.
Tecnologie Utilizzate

    CI/CD: Jenkins
    Containerizzazione: Docker
    Orchestrazione: Kubernetes
    Monitoraggio: Prometheus, Grafana
    Repository di Codice: GitHub
    Hosting: AWS (EKS)
